Installation:
-------------
Download and import the package into your unity project. When it's done, you are able to use all the features.

Quick start:
------------
1. Create an empty GameObject.
2. Add the component "TAF Planet Creator" to the empty GameObject.
3. Build your Planet as you like. :)
4. You can save the new planet as a prefab and use it again whenever you want.
